<---------Формат ввода данных--------->
[вещественное число] [вещественное число]

<---------Результат кластеризации--------->
Черный цвет - шум
Любой другой цвет - отдельный кластер

<---------Входные значения--------->
Адрес - Адрес файла с точками в заданном формате
K - целое неотрицательное количество кластеров для K-means (K > 0 и K больше чем количество точек)
Eps - вещественное расстояние для DBScan'а (Eps > 0)
MinPTS - целое неотрицательное число, минимальное количество точек в заданном радиусе для DBscan'a (MinPTS > 0)
T - целое число, количество точек в процентах, которые берутся от кластеров K-means(0 <= T <= 100)

<---------Алгоритм кластеризации K-means--------->
Алгоритм k-means  является простым повторяющимся алгоритмом кластеризации, 
который разделяет определенный набор данных на заданное пользователем число кластеров, k. 
Алгоритм прост для реализации и запуска, относительно быстрый, легко адаптируется и распространен на практике. 
Это исторически один из самых важных алгоритмов интеллектуального анализа данных.

<---------Алгоритм кластеризации DBScan--------->
Идея, положенная в основу алгоритма, заключается в том, что внутри каждого кластера наблюдается типичная плотность точек (объектов), 
которая заметно выше, чем плотность снаружи кластера,а также плотность в областях с шумом ниже плотности любого из кластеров. 
Ещё точнее, что для каждой точки кластера её соседство заданного радиуса должно содержать не менее некоторого числа точек, 
это число точек задаётся пороговым значением.

<---------Алгоритм кластеризации FastDBScan--------->
Вход: множество D, количество промежуточных кластеров для K-Means K, доля T;
Выход: кластеры и аномалии.
Шаг 1: инициализируем K центров случайно,
Шаг 2: реализуем алгоритм K-Means с K центрами,
Шаг 3: взяв T процентов каждого полученного кластера, получаем новое множе-
ство E,
Шаг 4: выполняем алгоритм DBSCAN с множеством E,
Шаг 5: отображаем обратно результаты, чтобы получить кластеры и аномалии для
множества D.

<---------Работа с графиком--------->
Колесико мыши - приближать отдалять
Выделение области левой кнопкой мыши - приближение именно этой области
На правую кнопку мыши - меню работы с графиком
